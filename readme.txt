Plugin to demonstrate NPF Framework

== Description ==

<h3>NPF Demo</h3>This plugin demonstrates the use of NPF Framework. NPF Framework is a framework for easily manage options for your plugin. It fully utilizes the Settings API of WordPress



== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'npf-demo.zip' from your computer
1. Upload
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'npf-demo.zip' to your computer
1. Upload the 'npf-demo' directory to your 'wp-content/plugins' directory
1. Activate the plugin on the WordPress Plugins dashboard
